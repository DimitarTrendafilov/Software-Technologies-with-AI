# Task Images

This folder contains images for task attachments. Images are categorized by task type:

## Image Categories

1. **requirements.jpg** - Documentation and requirements tasks
2. **design.jpg** - Design, mockups, and UI tasks
3. **development.jpg** - Development environment and coding tasks
4. **features.jpg** - Feature implementation tasks
5. **testing.jpg** - Testing and QA tasks
6. **documentation.jpg** - Documentation writing tasks
7. **deployment.jpg** - Deployment and infrastructure tasks
8. **monitoring.jpg** - Monitoring and support tasks
9. **performance.jpg** - Performance optimization tasks
10. **security.jpg** - Security audit tasks

## Placeholder Images

For seeding purposes, you can download free stock images from:
- Unsplash (https://unsplash.com)
- Pexels (https://pexels.com)
- Pixabay (https://pixabay.com)

Or generate placeholder images using:
- https://via.placeholder.com/800x600
- https://picsum.photos/800/600

## Usage

The seed script automatically uploads these images to Supabase Storage and attaches them to the last 5 tasks of the last 5 projects created.
