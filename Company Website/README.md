# Company Website

A professional company website with a clean and modern structure.

## Features

- **Responsive Header** - Navigation menu with links to all main sections
- **Main Content Area** - Organized sections for Home, About, Services, Products, and Contact
- **Footer** - Quick links, support information, social media connections, and copyright notice
- **Modern Design** - Clean layout with CSS styling and smooth transitions

## Structure

The website includes three main components:

### Header
- Company logo/branding
- Sticky navigation menu
- Responsive design

### Main Content
- Welcome section
- About section
- Services section
- Products section
- Contact section

### Footer
- Quick Links
- Support links
- Social media connections
- Copyright information

## Technologies Used

- HTML5
- CSS3
- Responsive Design

Simply open `index.html` in your web browser to view the website.
